Descrição

Base de dados de uma locadora de carros, contendo uma tabela para os dados dos clientes,
outra para os dados dos veículos e outra para os dados do aluguel em si.

Duas triggers que se propõe a atualizar os valores totais do aluguel, adicionando ou reduzindo
a quantidade de diárias.

Quatro views, sendo uma para diferir os veículos avariados dos não avariados, outra para
agrupar os ganhos da locadora em cada mês, outra para exibir o número de cada modalidade
de seguros que foram solicitados e ainda outra para determinar quantas locações foram feitas
em cada mês.
